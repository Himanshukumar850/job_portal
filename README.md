Job Portal Web Application
A full-stack Job Portal web application that enables both job seekers and recruiters to interact on a single platform. Users can search and apply for jobs, while recruiters can post and manage job listings efficiently.

Features

Job Seeker
 User Authentication (Signup/Login)
 Search and browse job listings
 Apply for jobs
 Profile management
 
Recruiter
 Post new job listings
 Edit and update job details
 Delete job postings
 Manage applications

  Tech Stack

Frontend:-
React.js
Tailwind CSS

Backend:-
Node.js
Express.js

Database:-
MongoDB

Installation & Setup

git clone https://github.com/Himanshukumar850/job_portal.git
cd job_portal
npm install
npm run dev
